Физика

Скорость \(ϑ = \frac{S}{t}\)
Пройденный путь \(S = ϑt\)
Время \(t = \frac{s}{ϑ}\)
Плотность \(ρ = \frac{m}{V}\)
Масса \(m = ρV\)
Объем \(V = \frac{m}{ρ}\)
Вес \(P = mg\)
Работа, механическая \(A = FS\)
Мощность \(N = \frac{A}{t}\)
Условие равновесия рычага \(\frac{F_1}{F_2} = \frac{l_1}{l_2}\)
Момент силы \(M = Fl\)
\(T = 2π\sqrt{\frac{l}{g}}\)
Условие равновесия рычага \(M_1 = M_2\)

Силы

Сила тяжести \(F_т=mg\)
Сила упругости \(F_упр=kΔl\)
Сила трения \(F_тр= μN\)
Сила Архимеда \(F_A= ρgV_т\)

Давление

Давление \(p = \frac{F}{S}\)
Давление жидкости \(p_ж= gh\)

Энергии

Кинетическая энергия \(E_к = \frac{mϑ^2}{2}\)
Потенциальная энергия \(E_p = mgh\)
Потенциальная энергия \(E_p = \frac{kx^2}{2}\)

Тепловые Явления

Нагревание тела \(Q = cm(t_1- t_2) = cmΔt\)
Сгорание тела \(Q = qm\)
Плавление тела \(Q = λm\)
Парообразование \(Q = Lm\)

Электричество

Сила тока \(I = \frac{q}{t}\)
Напряжение \(U= \frac{A}{|q|}\)
Сопротивление \(R = ρ \times \frac{l}{S}\)
Закон Ома \(I = \frac{U}{R}\)
КПД \(η = \frac{А_п}{А_з} \times 100%\)
Работа \(A = UIt\)
Мощность \(P = IU = \frac{U^2}{R} = I^2\)

Последовательное соединение

\(I_1 = I_2 = I_3\)
\(R = R_1 + R_2\)
\(U= U_1 + U_2\)

Параллельное соединение

\(U_1 = U_2 = U_3\)
\(I = I_1 + I_2\)
\(\frac{1}{R} = \frac{1}{R_1} + \frac{1}{R_2}\)

Передвижение

\(x = x_0 + S_x\)
\(S_x = x - x_0\)
\(a = \frac{ϑ - ϑ_0}{t}\)
\(S_x = ϑ_0x t + \frac{a_x t^2}{2}\)
\(S_x = \frac{ϑ_x^2 - ϑ_0x^2}{2a_x }\)
\(S_x = \frac{ϑ_x - ϑ_0x}{2 t}\)
